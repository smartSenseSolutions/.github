# SmartSense Solutions

## About This Repository

This `.github` repository holds organization-wide resources:

- `profile/README.md` powers the public SmartSense Solutions org profile
- Shared issue and pull request templates live under `.github/ISSUE_TEMPLATE/`
- Reusable GitHub Actions workflows are stored in `.github/workflows/`
- Automation configs such as `.github/dependabot.yml` keep dependencies current
